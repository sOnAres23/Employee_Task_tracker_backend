# Проект разработки сервиса трекера задач сотрудников с помощью Django REST Framework ^_^
## В данном проекте реализован функционал backend части приложения для создания сотрудников, задач и их интеграции с Телеграм Ботом, который реализует отправку сообщения о назначении задачи.

### Реализована модель Задачи, Сотрудника, Пользователя и соответствующие к ним контроллеры и сериализаторы, создана отложенная задача для отправки сообщений сотруднику через интеграцию с Telegram API.


## Установка:
1. клонируйте репозиторий через ssh 
```
git@github.com:sOnAres23/Employee_Task_tracker_backend.git
```
2. устанавливайте зависимости из файла pyproject.toml командой: 
```
poetry install
```

## Использование:
Проект на стадии разработки, для запуска приложения используется docker-compose Приложения собраны в контейнеры, конфигурация находится в файле: ```docker-compose.yml```

Запустить можно через команду:
```
docker-compose up
```
Или протестировать работоспособность можно локально через терминал, запустив:
```
python manage.py runserver
```
